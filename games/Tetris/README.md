# Enhanced Tetris

This is a significantly enhanced version of the classic Tetris game, featuring modern mechanics, visual effects, and a polished UI.

## Features

- **Game Modes**: Adjustable starting difficulty (Level 1, 5, 10, 15, 20).
- **Advanced Scoring**: Rewards for Single, Double, Triple, and Tetris line clears, plus Combo bonuses.
- **Hold Mechanic**: Press 'C' to store a piece for later use.
- **Visuals**: 3D block rendering, particle effects on clear, ghost piece, and animated score popups.
- **Statistics**: Tracks total lines, score, and breakdown of clear types.

## Controls

| Key | Action |
|-----|--------|
| **Left / Right** | Move piece |
| **Up** | Rotate clockwise |
| **Down** | Soft Drop |
| **Space** | Hard Drop (Instant) |
| **C** | Hold Piece |
| **P** | Pause |
| **R** | Restart / Menu |

## How to Run

From the root of the repository:

```bash
python game_launcher.py
```
Select "Tetris" from the launcher.

Alternatively, run directly from the terminal:

```bash
python games/Tetris/tetris.py
```
